# Cyclistic Bike-Share Analysis
### Google Data Analytics Capstone Project

## Project Overview
This project analyzes 2019-2020 Q1 trip data for Cyclistic, a fictional bike-share company. The goal is to identify behavioral differences between casual riders and annual members to drive conversion strategies.

## Data Process
- **Tools:** R (for cleaning/analysis), Tableau (for visualization).
- **Steps:** Merged 23 monthly datasets, cleaned trip duration, and categorized users.

## Key Insights
- Casual riders take longer trips (avg 60+ mins) primarily on weekends.
- Annual members are consistent weekday commuters with shorter trip times.
